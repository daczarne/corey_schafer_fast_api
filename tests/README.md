# Test plan

For each model that needs to be tested the plan is the following:

- Create item (`POST`)
  - create the item
  - get the item
- Update item (`PATCH`)
  - create the item
  - update the item
  - get the item
- List item(s) (`GET`)
  - create item(s)
  - list the item(s)
- Delete item (`DELETE`)
  - create item
  - delete item
  - get item
